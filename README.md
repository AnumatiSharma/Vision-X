# Vision-X
## AI-Powered Smart Vision Assistant for the Visually Impaired

Vision-X is an AI-powered assistive system designed to enhance the independence and safety of visually impaired individuals by detecting nearby objects in real time using computer vision and providing instant audio feedback.

##  Live Demo

https://vision-x-ai-pi.vercel.app/

##  Features

- Real-time object detection
- Voice-based audio feedback
- AI-powered computer vision
- Easy-to-use interface
- Accessibility-focused design

## 🛠️Technologies Used

- Python
- OpenCV
- YOLO
- HTML
- CSS
- JavaScript
